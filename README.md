-   nodejs
-   typescript
-   http (express default)
-   database (mongo default)

## Get Started

1. clone repository

```bash
$ git clone git@github.com:ayocodingit/clean-architecture-ts.git
```

2. install node

```bash
$ npm install
```

3. Copy `.env` from `.env.example`

```bash
$ cp .env.example .env
```
