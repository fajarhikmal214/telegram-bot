-   nodejs
-   typescript
-   database (mongo default)

## Get Started

1. clone repository

```bash
$ https://github.com/fajarhikmal214/telegram-bot
```

2. install node

```bash
$ npm install
```

3. Copy `.env` from `.env.example`

```bash
$ cp .env.example .env
```
